# hr

This repository contains the early work-in-progress for the project and is published publicly so others can view its development.

## Repository Structure

- `web-app/`: TanStack-based web application
- `database/`: local database setup
- `video-processing-service/`: service that reacts to uploaded videos and transforms them to DASH
- `local-s3-compatible-storage/`: local SeaweedFS-based S3-compatible storage for development
